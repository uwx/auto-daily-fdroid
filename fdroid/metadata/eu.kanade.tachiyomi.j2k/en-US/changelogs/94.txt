**Note:** This release has builds separated by device's architecture, if you're on a version older than v1.4.5, you will need to manually download the universal apk from the release page
  
## Additions
* Added Lavender (@crepetf) / Violet (@osyx) theme 
* When "show all categories" is disabled, you can now swipe horizontally to change categories
* Migration list updates
  * Migration list (in browse) now shows language + number of entries in each source
  * Option to sort alphabetically (default), by most entries, or by obsolete/uninstalled sources
  * Show if a source is obsolete or uninstalled in the extension list
* Option to group library by author/artist
* Add fallback for GitHub extensions (@arkon, @stevenyomi)

## Changes
* Update WebView user agent (@arkon)
* Optimize split tall images (@antsylich)
* Fast scroll in library now snaps to category headings
* Rename navigation layout to tapzones (@arkon)
* Mangaupdates fixes and improvements (@ghostbear)
  * Fixed certain titles not being decoded properly
  * Added decimal scoring
* Translations have been updated (To help translate, you can [contribute more to Tachiyomi or TachiJ2K here](https://hosted.weblate.org/engage/tachiyomi/?utm_source=widget))

## Fixes
* Fixed collapsed categories ignoring filters
* Fixed searching collapsed categories when using single category mode


**Full Changelog**: https://github.com/Jays2Kings/tachiyomiJ2K/compare/v1.5.5...v1.5.6
